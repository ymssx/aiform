AI Form Helper
v1.0.0
An AI-powered Chrome extension for intelligent form data extraction and auto-filling.
✨ Features
  • Smart Fill — AI analyzes DOM to auto-fill forms with profile data & memories
  • Data Extraction — Captures form submissions and builds structured user profiles
  • Memory System — Remembers preferences, plans, and facts across sessions
  • Component Libraries — Supports Ant Design, Element UI, TDesign, MUI, and more
  • Multi-Backend — Works with Qwen, DeepSeek, OpenAI, Ollama, and any OpenAI-compatible API
📦 Tech Stack
• Chrome Extension Manifest V3
• Vanilla JavaScript (no framework dependency)
• OpenAI-compatible Chat Completions API
• Pluggable component adapter architecture
